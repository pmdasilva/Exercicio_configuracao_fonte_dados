# Exercicio_configuracao_fonte_dados

### Estudo de caso
Jamie, um executivo da Adventure Works, pediu que o senhor preparasse um relatório para o departamento de vendas, exibindo os detalhes de qualquer pedido necessário, incluindo o produto, o preço unitário, a quantidade do pedido e a taxa de desconto para a reunião de revisão anual. Cada gerente de país compartilhou uma pasta de trabalho do Microsoft Excel contendo os dados necessários. Cabe ao senhor conectar-se às fontes de dados e combiná-las em um único relatório!

### Etapa 1: Fazer download e preparar o arquivo

* A primeira etapa é garantir que o arquivo do Excel esteja salvo no seu computador ou que o senhor tenha acesso a ele por meio do OneDrive for Business, contas pessoais do OneDrive ou SharePoint Team Sites. Certifique-se de que seus arquivos atendam aos seguintes requisitos:

* O conjunto de dados é fornecido como um arquivo simples: Sua pasta de trabalho não deve ter nenhum total de linhas ou colunas.

* Todos os dados em cada coluna devem ser do mesmo tipo: Quer se trate de datas, texto ou moeda, cada coluna deve conter o mesmo tipo de dados.

* O arquivo está em um formato de tabela facilmente legível. Ele não contém tabelas dinâmicas ou formatos de matriz

* Um nome sem espaço: Dê à sua tabela um nome fácil de lembrar. Lembre-se de que o nome não pode conter espaços.

### Etapa 2: obter dados

* O Power BI Desktop oferece a capacidade de se conectar a várias fontes de dados, desde fontes básicas, como arquivos do Microsoft Excel, até fontes mais complexas.

* Como a tarefa que lhe foi atribuída é conectar-se às fontes de dados necessárias, vamos nos conectar a elas. Escolha o tipo de dados necessário na lista.

* Navegue até o local onde o arquivo do Excel está salvo.

* Escolha a(s) planilha(s) e a(s) tabela(s) que deseja importar na janela Navigator.

* Clique em Load (Carregar). Isso importará seus dados do Excel para o Power BI.

Dica: o senhor pode inspecionar os dados na exibição de tabela (também conhecida como exibição de dados) clicando no ícone de tabela na barra lateral esquerda.

Salve seu projeto do Power BI.
